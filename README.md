# 💰 WealthOS

> The Operating System for Your Wealth

WealthOS is a modern personal wealth management platform built with Next.js, designed to help users track investments, analyze portfolio performance, monitor live market prices, and make smarter financial decisions.

---

## ✨ Features

### Dashboard
- Portfolio Overview
- Net Worth
- Today's P/L
- Portfolio Chart
- Asset Allocation
- Live Gold Price
- Live Silver Price

### Portfolio
- Gold
- Silver
- Stocks
- ETFs
- Mutual Funds
- Debt
- Cash

### Analytics
- Asset Allocation
- Performance Charts
- CAGR (Planned)
- XIRR (Planned)
- Portfolio Health Score (Planned)

### AI (Planned)
- AI Portfolio Assistant
- Daily Briefing
- Investment Suggestions
- Goal Tracking
